## Getting Started

### Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

### Build

```bash
npm run build
npm start
```

## Usage

1. **Set Targets**: Go to Settings to set your daily calorie and carb targets
2. **Track Daily**: Click on any day in the calendar to log your intake
3. **View History**: Check the History tab to review past entries
4. **Manage Data**: Export/import your data from the Settings page

## PWA Installation

The app can be installed on your mobile device:

1. Open the app in your mobile browser
2. Tap the "Add to Home Screen" option
3. The app will be installed and available offline
